## The operating rules (non-negotiable, every seat)
- **Routing (who does what):** Creative Director (Iris) **directs** → the brand window + B&T **execute** → Creative Director **gates**. Seats stay in lane: CMO sets standards + gates + routes (does NOT do brand execution); the brand window owns its surface; B&T builds; Iris owns visual quality cross-brand.
- **The gates (pre-publish):** hero-metrics (NO fabricated/unsourced stats), no em-dashes in outbound, per-brand voice + correct on-camera face (Michael = AvI/WD/AIPG/BAE, Ryan = Book'd, Miriam = P&P, never crossed), claims checked against the ledger, client output gated on the client's approval.
- **The bar:** better than European / Fifth Avenue, applied to each brand's OWN truth, not generic luxury cosplay.
- **Prompt-economy / first-pass yield:** tight briefs + locked references, generation does not spray-and-pray; cost-per-approved-asset falls over time.
- **Cross-chat work routes via flags** (`🏁 FLAG FOR:` in the owning seat's state file → router → Slack). Don't copy docs between chats; route the work, the doc lives in the repo.

## How the bible stays current
Any new standing standard (a new system, template, persona, or rule) gets a one-line entry in the table above when it is created. The bible is the index; the files are the depth. If it is not here, seats should not treat it as canon.
