## The Core Principle

Sofia writes **one anchor asset** per week. Everything else is a derivative — no original thinking required, only reformatting. One hour of thinking → eight hours of distribution.

**Anchor types** (rotating weekly):
1. **Long-form blog** (1,200–1,800 words, SEO-targeted)
2. **Case study** (client result + methodology)
3. **Opinion piece** (contrarian POV on digital marketing / AI)
4. **Tactical guide** (step-by-step how-to)

## Weekly Production Rhythm

**Monday — Write the anchor (2 hours)**
Sofia writes the long-form piece. Full research, full edit, publish-ready. This is the only net-new creative work of the week.

**Tuesday — Derivative batch #1 (2 hours)**
Items 1–5 (all LinkedIn + IG + X). Use the anchor as source material; reformat, don't rewrite.

**Wednesday — Derivative batch #2 (1.5 hours)**
Items 6–10 (video scripts + sales enablement). These are shorter and largely copy-paste with angle shifts.

**Thursday — Schedule + personal posts**
All 10 derivatives scheduled in Buffer/Later. Dek's personal LinkedIn gets a paraphrased version of item 1.

**Friday — Engagement block (1 hour)**
Reply to every comment on that week's content. Engagement drives reach more than publishing does.

## Derivative Templates

### Template 1: LinkedIn Text Post (from blog)

```
Hook (1 line, screen-stopping contrarian or data point)
[empty line]
Context (2 lines — who this is for + why it matters)
[empty line]
5–7 bullet points (the blog's core ideas, compressed)
[empty line]
CTA (link to blog OR engagement question)
```

**Example hook starters:**
- "Most {{industry}} business owners are doing {{thing}} wrong. Here's what actually works:"
- "I audited 47 {{industry}} websites this month. 44 made the same mistake:"
- "Stop {{common advice}}. Start {{contrarian advice}}. Here's why:"

### Template 2: Twitter Thread

```
Tweet 1: Hook + "🧵" (or number the thread)
Tweet 2: Context + promise
Tweets 3–8: One point per tweet (same points as LinkedIn, fewer words)
Final tweet: CTA + link to blog
```

### Template 3: IG / LinkedIn Carousel

```
Slide 1: Bold statement / contrarian hook (big text, minimal)
Slide 2: "Here's why this matters" (set up the problem)
Slides 3–7: One idea per slide with a visual / icon
Slide 8: CTA — "Save this post" or "DM us 'AUDIT' for a free review"
```

### Template 4: 30-Second Video Script

```
[0–3 sec]   Pattern interrupt — the hook
[3–10 sec]  The problem
[10–20 sec] The insight / framework
[20–28 sec] The payoff / example
[28–30 sec] CTA — "follow for more" or "comment AUDIT"
```

### Template 5: Newsletter Snippet

```
Subject: [Intriguing question from the anchor]
Body:
  - 2 sentence setup
  - 3 bullet takeaways from the anchor
  - Link: "Read the full breakdown ({{time}} min read)"
  - P.S. — soft CTA (free audit / reply offer)
```

## Batching Tricks (cut production time 60%)

1. **Write the hook bank first**: Before writing the anchor, brainstorm 10 possible hooks. Use the best one for the blog intro; reuse the other 9 as hooks for derivatives.

2. **The "points list"**: Turn the anchor's structure into a numbered list of 5–7 core points. This single list becomes the skeleton for LinkedIn, Twitter, carousels, and video scripts.

3. **Voice variants cheat sheet**:
   - Blog = professional, full sentences
   - LinkedIn = punchy, one idea per line
   - Twitter = even shorter, sharper edges
   - IG = visual, minimal words
   - Video = conversational, spoken-not-written

4. **AI-assisted reformatting**: Use a single Claude prompt to convert the anchor into 5 of the formats in one shot. Sofia reviews and edits, never drafts from scratch.
